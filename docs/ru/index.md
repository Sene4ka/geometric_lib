# geometric lib

## Общее описание
geometric_lib - библитотека, реализующая различные функции для вычисления параметров геометрических фигур

## Документация функций реализованных фигур:
* [Круг](./circle.md)
* [Квадрат](./square.md)
* [Прямоугольник](./rectangle.md)
* [Треугольник](./triangle.md)
